es6新增的let声明方式，专用于变量的声明，let有了块级作用域的概念。

它只在其声明的作用域起作用，超出范围的变为‘undefined'

不存在变量提升，如果未声明就使用会报错

if和for这种都存在自己的作用域，不再是同一window作用域下的了

如果是从未生命过的变量typeof不会报错，但如果是声明在typeof执行之后就会报错

暂时性死区：let x = x;会报错，不能够在声明前使用，因为执行顺序是先右后左

不允许在相同作用域内，重复声明同一个变量。 function a (arg){ let arg ;//报错 } function a(arg){ { let arg;//不报错，不是同一个作用域了。 } }
